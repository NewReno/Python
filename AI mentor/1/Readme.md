# Задачи для повторения и закрепления основ Python

## 1. Переменные и типы данных
### Лёгкий уровень
**Задача 1**  
Напишите программу, которая запрашивает у пользователя температуру в градусах Цельсия и переводит её в Фаренгейты по формуле:  
`F = (C * 9/5) + 32`. Выведите результат с помощью f-строки.

**Задача 2**  
Создайте переменные `name` (строка), `age` (число), `is_student` (логическое значение). Выведите их значения в формате:  
`"Имя: {name}, Возраст: {age}, Студент: {is_student}"`.

---

### Средний уровень
**Задача 3**  
Напишите программу, которая принимает строку от пользователя и выводит:  
- Количество символов в строке.  
- Первый и последний символы.  
- Строку в верхнем регистре.

**Задача 4**  
Проверьте, является ли введённая строка числом (целым или дробным). Если да — выведите его квадрат, иначе — сообщение об ошибке.

---

### Сложный уровень
**Задача 5**  
Создайте список `data`, содержащий целое число, строку, `None` и список.  
Пройдитесь по элементам циклом и для каждого выведите его тип.  
Если элемент — `None`, замените его на строку `"Ничего нет"`.

---

## 2. Условные операторы и циклы
### Лёгкий уровень
**Задача 1**  
Напишите программу, которая определяет, является ли число чётным или нечётным.

**Задача 2**  
Попросите пользователя ввести год и определите, является ли он високосным (делится на 4, но не на 100, или делится на 400).

---

### Средний уровень
**Задача 3**  
Напишите программу, которая выводит сумму всех чисел от 1 до `n` (включительно), где `n` вводит пользователь.

**Задача 4**  
Проверьте, является ли введённое число простым (делится только на 1 и само себя).  
Используйте цикл `for` и оператор `break` для оптимизации.

---

### Сложный уровень
**Задача 5**  
Реализуйте игру "Угадай число":  
- Программа загадывает число от 1 до 100 (используйте `random.randint`).  
- Пользователь вводит варианты, пока не угадает.  
- После каждого ввода программа подсказывает, больше или меньше число.

---

## 3. Функции
### Лёгкий уровень
**Задача 1**  
Напишите функцию `calculate_area`, которая принимает длину и ширину прямоугольника и возвращает его площадь.

**Задача 2**  
Создайте функцию `greet`, которая принимает параметр `name` (по умолчанию — `"Гость"`) и выводит:  
`"Привет, {name}!"`.

---

### Средний уровень
**Задача 3**  
Напишите функцию `sum_numbers`, которая принимает любое количество чисел через `*args` и возвращает их сумму.

**Задача 4**  
Создайте функцию `print_info`, которая принимает `**kwargs` и выводит их в формате:  
`"{ключ}: {значение}"` для каждого элемента.

---

### Сложный уровень
**Задача 5**  
Реализуйте рекурсивную функцию `fibonacci`, которая возвращает n-е число Фибоначчи.  
Пример: `fibonacci(5)` → `5` (ряд: 0, 1, 1, 2, 3, 5).

---

## Подсказки
- Используйте `type()` для проверки типов данных.  
- В сложных задачах разбивайте код на вспомогательные функции.  
- Для работы с `None` применяйте условие `if x is None`.
