# План обучения программированию на Python

## 1. Повторение и закрепление основ
- **Переменные, типы данных**: Убедись, что свободно работаешь с `int`, `float`, `str`, `bool`, `None`.
- **Условные операторы и циклы**: Практикуй сложные условия (`and`, `or`, `not`), вложенные циклы, `break`/`continue`.
- **Функции**: Изучи параметры по умолчанию, `*args`, `**kwargs`, область видимости (`global`, `nonlocal`), рекурсию.
- **Пример задачи**: Напиши функцию, которая проверяет, является ли число простым.

---

## 2. Структуры данных
- **Списки (list)**: Методы (`append`, `extend`, `pop`, `sort`), списковые включения.
- **Кортежи (tuple)**: Неизменяемость, использование в качестве ключей словаря.
- **Словари (dict)**: Методы (`get`, `items`, `update`), словарные включения.
- **Множества (set)**: Операции (`union`, `intersection`, `difference`).
- **Пример задачи**: Подсчитай частоту слов в тексте с помощью словаря.

---

## 3. Объектно-ориентированное программирование (ООП)
- **Классы и объекты**: Атрибуты, методы, `__init__`.
- **Наследование**: Переопределение методов, `super()`.
- **Инкапсуляция**: Приватные атрибуты (`_x`, `__x`).
- **Полиморфизм**: Использование общих интерфейсов.
- **Магические методы**: `__str__`, `__repr__`, `__add__`.
- **Пример задачи**: Создай класс `BankAccount` с методами пополнения, снятия средств и проверки баланса.

---

## 4. Работа с файлами и исключения
- **Чтение/запись файлов**: `open()`, режимы (`r`, `w`, `a`, `b`), менеджер контекста `with`.
- **Обработка исключений**: `try/except/else/finally`, создание собственных исключений.
- **Пример задачи**: Напиши скрипт, который читает CSV-файл и выводит сумму чисел в столбце.

---

## 5. Модули и библиотеки
- **Стандартная библиотека**:
  - `os`/`sys` — работа с системой.
  - `datetime`/`time` — дата и время.
  - `json`/`pickle` — сериализация данных.
  - `collections` — `namedtuple`, `defaultdict`, `Counter`.
- **Виртуальные окружения**: `venv`, установка пакетов через `pip`.
- **Популярные библиотеки** (на выбор):
  - **Для анализа данных**: `NumPy`, `Pandas`.
  - **Для веб-разработки**: `Flask`, `Django`.
  - **Для автоматизации**: `BeautifulSoup`, `Selenium`.
- **Пример задачи**: С помощью `requests` и `BeautifulSoup` собери данные с веб-страницы.

---

## 6. Практические проекты
Начни с малого и постепенно усложняй:
1. **Консольные приложения**:
   - Калькулятор.

4. **Работа с данными**:
   - Анализ CSV-файла с помощью `Pandas`.
5. **Автоматизация**:
   - Скрипт для переименования файлов в папке.
